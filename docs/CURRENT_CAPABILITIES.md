# 📋 Возможности текущего состояния платформы Sadara

## 📊 Обзор платформы

Платформа Sadara — это полноценная система управления подключениями к сети и клиентами для ISP-компаний. В текущем состоянии она предоставляет широкий спектр функций, но имеет ряд областей для улучшения.

---

- ✅ Двухфакторная аутентификация
- ✅ Автоматическое выход из системы по времени бездействия
