## 🏗️ المعمارية التقنية

### التطبيقات (Clients)
| التطبيق | التقنية | الغرض |
|---------|---------|-------|
| Desktop App | Flutter Windows | إدارة الشركات |
| Citizen Web | Next.js | بوابة المواطن |
| Citizen Mobile | Flutter | تطبيق المواطن |
| Technician Mobile | Flutter | تطبيق الفني |
| Admin Panel | Next.js | لوحة Super Admin |

### الخدمات الخلفية (Backend Services)
| الخدمة | الوظيفة |
|--------|---------|
| Auth Service | المصادقة (JWT, OAuth, OTP) |
| Users Service | إدارة المستخدمين |
| Companies Service | إدارة الشركات |
| Service Requests | طلبات الخدمة |
| Tracking Service | تتبع GPS |
| Chat Service | المحادثات |
| Notification Service | الإشعارات |
| Payment Service | المدفوعات |
| Shop Service | المتجر |

### قواعد البيانات
| قاعدة البيانات | الاستخدام |
|----------------|----------|
| PostgreSQL | البيانات الرئيسية |
| Redis | Cache + Sessions + Online Status |
| MinIO/S3 | الملفات والصور |
| InfluxDB | بيانات GPS والتحليلات |

### Real-time Layer
- **SignalR Hub** للاتصال الفوري
  - Tracking Hub (مواقع الفنيين)
  - Chat Hub (الرسائل)
  - Notifications Hub (الإشعارات)

## 📊 جداول قاعدة البيانات الجديدة

### جدول المواطنين (citizens)
```sql
CREATE TABLE citizens (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES users(id),
    national_id VARCHAR(20),
    address TEXT,
    city VARCHAR(50),
    preferred_company_id UUID REFERENCES companies(id),
    wallet_balance DECIMAL(10,2) DEFAULT 0
);
```

### جدول طلبات الخدمة (service_requests)
```sql
CREATE TABLE service_requests (
    id UUID PRIMARY KEY,
    citizen_id UUID REFERENCES citizens(id),
    company_id UUID REFERENCES companies(id),
    service_type VARCHAR(50),
    status VARCHAR(20),
    assigned_technician_id UUID REFERENCES users(id),
    details JSONB,
    created_at TIMESTAMP DEFAULT NOW()
);
```

### جدول مواقع الفنيين (technician_locations)
```sql
CREATE TABLE technician_locations (
    id UUID PRIMARY KEY,
    technician_id UUID REFERENCES users(id),
    service_request_id UUID REFERENCES service_requests(id),
    latitude DECIMAL(10, 8),
    longitude DECIMAL(11, 8),
    accuracy DECIMAL(5, 2),
    speed DECIMAL(5, 2),
    recorded_at TIMESTAMP DEFAULT NOW()
);
```

### جدول غرف المحادثة (chat_rooms)
```sql
CREATE TABLE chat_rooms (
    id UUID PRIMARY KEY,
    service_request_id UUID REFERENCES service_requests(id),
    room_type VARCHAR(20),
    created_at TIMESTAMP DEFAULT NOW(),
    is_active BOOLEAN DEFAULT true
);
```

### جدول الرسائل (chat_messages)
```sql
CREATE TABLE chat_messages (
    id UUID PRIMARY KEY,
    room_id UUID REFERENCES chat_rooms(id),
    sender_id UUID REFERENCES users(id),
    message_type VARCHAR(20),
    content TEXT,
    media_url VARCHAR(500),
    latitude DECIMAL(10, 8),
    longitude DECIMAL(11, 8),
    is_read BOOLEAN DEFAULT false,
    created_at TIMESTAMP DEFAULT NOW()
);
```

### جدول الإشعارات (notifications)
```sql
CREATE TABLE notifications (
    id UUID PRIMARY KEY,
    user_id UUID REFERENCES users(id),
    notification_type VARCHAR(50),
    title VARCHAR(200),
    body TEXT,
    data JSONB,
    is_read BOOLEAN DEFAULT false,
    is_sent BOOLEAN DEFAULT false,
    sent_via JSONB,
    created_at TIMESTAMP DEFAULT NOW(),
    read_at TIMESTAMP
);
```
