## 🏗️ البنية العامة

```
المشروع يستخدم Clean Architecture:

┌─────────────────────────────────────┐
│           API (Controllers)          │  ← نقطة الدخول
├─────────────────────────────────────┤
│        Application (Services)        │  ← منطق العمل
├─────────────────────────────────────┤
│           Domain (Entities)          │  ← الكيانات
├─────────────────────────────────────┤
│     Infrastructure (DB, External)    │  ← قاعدة البيانات
└─────────────────────────────────────┘
```

2. **جرب الكود** - شغّل وعدّل واختبر

4. **اسأل عن أي شيء** - لا تتردد في السؤال
