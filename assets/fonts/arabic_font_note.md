# خط النصوص العربية

يرجى تحميل خط عربي (مثل Noto Sans Arabic) من Google Fonts وحفظه باسم:
- NotoSansArabic-Regular.ttf

أو يمكن استخدام الخط المدمج من Google Fonts مباشرة.

لتحميل الخط:
1. اذهب إلى https://fonts.google.com/noto/specimen/Noto+Sans+Arabic
2. حمل الخط 
3. ضعه في مجلد assets/fonts/
4. أو اتركه فارغ ليستخدم التطبيق PdfGoogleFonts تلقائياً
