## 📋 نظرة عامة

تطبيق الصدارة هو حل شامل ومتطور لإدارة شبكات FTTH (Fiber to the Home) مصمم خصيصاً للشركات التقنية في المنطقة العربية. يوفر التطبيق واجهة سهلة الاستخدام لإدارة المشتركين والباقات والمناطق مع دعم كامل للغة العربية.

## 🛠️ المتطلبات التقنية

### البرمجيات المطلوبة
- Flutter SDK (آخر إصدار)
- Dart SDK
- Android Studio / VS Code

### الحزم المستخدمة
```yaml
dependencies:
  flutter:
    sdk: flutter
  http: ^0.13.5
  shared_preferences: ^2.0.15
  intl: ^0.17.0
  geolocator: ^9.0.2
  googleapis: ^9.2.0
  googleapis_auth: ^1.4.1
```

### المتطلبات الخارجية
- **Google Sheets API**: لتسجيل العمليات
- **ملف المصادقة**: `assets/service_account.json`
- **اتصال إنترنت**: مطلوب للتشغيل

## 🔧 التثبيت والتشغيل

### 1. تحضير البيئة
```bash
# تثبيت Flutter dependencies
flutter pub get

# فحص متطلبات النظام
flutter doctor
```

### 2. إعداد Google Sheets API
1. إنشاء مشروع في Google Cloud Console
2. تفعيل Google Sheets API
3. إنشاء Service Account وتحميل المفتاح
4. وضع الملف في `assets/service_account.json`

### 3. تشغيل التطبيق
```bash
# تشغيل في وضع Debug
flutter run

# بناء للإنتاج
flutter build apk --release
```
